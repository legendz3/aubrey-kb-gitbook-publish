# Getting Started

Start here if you are setting up Aubrey for the first time or learning how agents work.

## Articles in this section

* [Getting Started with Aubrey](getting-started-with-aubrey.md): create an account, complete onboarding, and create your first agent.
* [Terminology](terminology.md): learn what Aubrey means by agents, workspace, recipes, skills, channels, and integrations.
* [Creating and Managing Agents](creating-and-managing-agents.md): create another agent, switch between agents, and understand workspace tabs.
* [Chatting with Aubrey](chatting-with-aubrey.md): send messages, attach files, retry runs, and respond to approval cards.

## Good first step

If you are not sure where to begin, open [Getting Started with Aubrey](getting-started-with-aubrey.md) and follow the setup flow from account creation through the first agent preview.
