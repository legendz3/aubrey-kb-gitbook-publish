# Settings

Use Settings when you need to manage your account, billing, plan usage, or optional quality data preferences.

## Articles in this section

* [Billing and Account Settings](billing-and-account-settings.md): update account details, check plan usage, manage billing, and set quality data preferences.

## Good first step

Open **Settings** in Aubrey, then choose **Account**, **Billing**, or **Quality** depending on what you need to change.
